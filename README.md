## Installation

In R, run:
```r
> ### For windows binary and Rgui.
> install.packages("https://github.com/snoweye/IRkernel-win/raw/master/IRkernel_0.7.1.zip")
>
> ### For source.
> install.packages("https://github.com/snoweye/IRkernel-win/raw/master/IRkernel_0.7.1.tar.gz")

```
